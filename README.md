# ✨ Task Manager

> A sleek, dark-themed task manager built with Flask.

## 🎯 Features

- 📝 **Add /** 🗑️ **Delete tasks**
- 🔴 High · 🟡 Medium · 🟢 Low priorities
- 🔍 **Filter:** All · Active · Completed
- 📊 **Sort:** Priority · Date · Alphabet
- 🌙 **Dark theme** 
- 💻 **Desktop app mode**

## 🚀 Quick Start

```bash
git clone https://github.com/R1sen13/Task_Manager.git
cd Task_Manager
pip install flask pywebview
python launcher.py
```
## 📡 API

`GET http://127.0.0.1:5000/api/tasks`
